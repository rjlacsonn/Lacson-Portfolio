HTML Lists

HTML lists help organize information effectively. There are three types:

  • Ordered Lists <ol>
  • Unordered Lists <ul>
  • Description Lists <dl>

Ordered Lists

<ol>
  <li>Notebook</li>
  <li>Ballpen</li>
  <li>Yellow Paper</li>
</ol>
li {
  list-style-type: upper-roman;
}

List Style Types

StyleDescription
decimalNumbers (default)
upper-alphaUppercase letters
lower-alphaLowercase letters
upper-romanUppercase Roman numbers
lower-romanLowercase Roman numbers

Unordered List

<ul style="list-style-type:square">
  <li>Notebook</li>
  <li>Ballpen</li>
  <li>Yellow Paper</li>
</ul>

Bullet Styles

ValueDescription
discDefault bullet
circleHollow circle
squareSquare bullet
noneNo bullet

Description List

<dl>
  <dt>Notebook</dt>
  <dd>- a small book for notes</dd>
  <dt>Ballpen</dt>
  <dd>- a pen with ink</dd>
</dl>

Nested Lists

<ul>
  <li>Coffee</li>
  <li>Tea
    <ul>
      <li>Black tea</li>
      <li>Green tea</li>
    </ul>
  </li>
  <li>Milk</li>
</ul>

Horizontal List (Navigation)

<style>
ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  overflow: hidden;
  background-color: #333;
}
li {
  float: left;
}
li a {
  display: block;
  color: white;
  text-align: center;
  padding: 16px;
  text-decoration: none;
}
li a:hover {
  background-color: #111;
}
</style>

HTML Links / Hyperlinks

The <a> (anchor) tag creates links to other pages.

<a href="home.html">Back to HOME</a>
<a href="http://google.com">Google</a>
<a href="template.htm">template</a>

Target Attribute

ValueDescription
_blankNew tab or window
_selfSame tab (default)
_parentParent frame
_topFull window

Image Links

<a href="template.html">
  <img src="panda.png" alt="Lect1" style="width:42px;height:42px;border:0;">
</a>

Floating Images

<p><img src="polar.png" style="float:right;width:42px;height:42px;">Text beside image.</p>

Image Maps

Use the usemap and <map> tags to create clickable hotspots on an image.

<map name="map-name">
  <area shape="rect" coords="34,44,270,350" href="link.html" alt="Rect">
</map>

Shape Types

  • rect
  • circle
  • poly
  • default

HTML Images

Images are added using <img> tag with attributes like src and alt.

<img src="panda.png" alt="Panda Bear" style="width:500px;height:600px;">
<img src="images/circle.jpg" alt="Circle" style="width:128px;height:128px;">
<img src="https://www.ue.edu.ph/manila/main/misc/ccss.png" alt="UE-CCSS">
