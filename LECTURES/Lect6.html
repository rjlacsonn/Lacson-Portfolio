Web Forms

Web forms are interactive elements that allow user input and data submission. Common controls include:

  • Text inputs
  • Radio buttons
  • Checkboxes
  • Select dropdowns
  • Textareas
  • Color pickers, sliders, calendars

The <form> Tag

Attributes: action, method, target, enctype, etc.

<form action="/action_page.php" method="post" target="_blank">
  ...
</form>

The <input> Element

Defines input controls. Types include:

  • text
  • password
  • radio
  • checkbox
  • submit, reset, button
  • color, date, email, number, range, url, etc.
<input type="text" id="fname" name="fname" placeholder="Enter name">

Other Form Elements

<label>

Associates a text label with an input:

<label for="fname">First Name:</label>

<select> and <option>

<select name="movies" id="movies">
  <option value="Ten">Ten Things I Hate About You</option>
  ...
</select>

<textarea>

<textarea name="message" rows="4" cols="50">
Your message here.
</textarea>

<fieldset> and <legend>

<fieldset>
  <legend>Personal Info</legend>
  <label>Name:</label>
  <input type="text" name="name">
</fieldset>

Form Attributes

  • value, readonly, disabled
  • size, maxlength
  • autocomplete, placeholder, required

Creating Effective Forms

  • Use appropriate field sizes
  • Group inputs logically
  • Use consistent labels and help text

HTML Multimedia

Includes sound, video, and interactive content on web pages.

The <audio> Tag

<audio controls>
  <source src="sound.mp3" type="audio/mpeg">
  Your browser does not support audio.
</audio>

The <video> Tag

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support video.
</video>

Autoplay Example

<video autoplay>
  <source src="auto.mp4">
</video>

HTML Plugins

Embedded objects via <object> or <embed>.

<object width="400" height="200" data="file.swf"></object>
<embed width="400" height="200" src="file.swf">
