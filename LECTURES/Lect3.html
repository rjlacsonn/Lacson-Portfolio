HTML Styles

The style attribute is used to add styles such as color, font, size, and more to an element.

<tagname style="property:value;">

Background Color

<body style="background-color:DarkTurquoise;">
<h1 style="background-color:cyan;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

Text Color

<h1 style="color:red;">This is a heading</h1>
<p style="color:blue;">This is a paragraph.</p>

Fonts

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

Text Size

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:30pt;">This is a paragraph.</p>

Text Alignment

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

HTML Tables

An HTML table is defined with the <table> tag. Rows use <tr>, headers use <th>, and data uses <td>.

<table style="width:100%">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Nicole</td>
    <td>Pascual</td>
    <td>18</td>
  </tr>
  <tr>
    <td>Allen</td>
    <td>Ramirez</td>
    <td>18</td>
  </tr>
</table>

Table Border

table, th, td {
  border: 1px solid black;
}

Collapsed Borders

table, th, td {
  border: 1px solid black;
  border-collapse: collapse;
}

Cell Padding

th, td {
  padding: 15px;
}

Left-Align Headings

th {
  text-align: left;
}

Border Spacing

table {
  border-spacing: 5px;
}

Spanning Cells

Colspan

<table style="width:100%">
  <tr>
    <th>Company Name</th>
    <th colspan="2">Telephone</th>
  </tr>
  <tr>
    <td>The Travel Club</td>
    <td>(02) 332-7183</td>
    <td>(02) 815-7418</td>
  </tr>
</table>

Rowspan

<table style="width:100%">
  <tr>
    <th>Company Name:</th>
    <td>The Travel Club</td>
  </tr>
  <tr>
    <th rowspan="2">Telephone:</th>
    <td>(02) 332-7183</td>
  </tr>
  <tr>
    <td>(02) 815-7418</td>
  </tr>
</table>
