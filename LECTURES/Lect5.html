Introduction to CSS3

CSS (Cascading Style Sheets) separates structure from presentation and allows consistent styling across web pages. CSS3 introduces modules, backwards compatibility with CSS2.1, and enhanced styling capabilities.

CSS Syntax and Selectors

selector {
  property: value;
}

Selectors

  • Element Selector
  • ID Selector: #idname
  • Class Selector: .classname
  • Grouping Selectors: h1, h2, p {}

CSS Comments

/* This is a comment */

Ways to Insert CSS

  • External Style Sheet
  • Internal Style Sheet
  • Inline Style
<link rel="stylesheet" type="text/css" href="mystyle.css">

Cascading Order

  1. Inline
  2. Internal
  3. External
  4. User Settings
  5. Browser Defaults

CSS Colors

background-color: rgba(255, 0, 0, 0.5);

Color formats include RGB, HEX, HSL, RGBA, HSLA.

CSS Backgrounds

body {
  background-image: url("img_tree.png");
  background-repeat: no-repeat;
  background-position: right top;
  background-attachment: fixed;
}

Use shorthand: background: #fff url(...) no-repeat right top;

CSS Borders

  • border-style: dotted, dashed, solid, etc.
  • border-width: thin, medium, thick
  • border-color: name, hex, rgb
  • Shorthand: border: 5px solid red;
  • Individual sides: border-left:, border-bottom:
  • Rounded: border-radius: 5px;

CSS Dimensions

Use height, width, and max-width to control sizing.
